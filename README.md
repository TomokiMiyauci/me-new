## Getting Started

### Requirements

Deno >= 2

### Installation

Clone the repository and install dependencies:

```bash
deno install
```

### Development

Start the development server:

```bash
deno task dev
```

### Build

Build the application for production:

```bash
deno task build
```

### Preview

Preview the production build locally:

```bash
deno task preview
```

### Available Tasks

Check all available tasks:

```bash
deno task
```
